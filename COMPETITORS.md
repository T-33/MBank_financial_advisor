# Competitor Analysis

## Western (Europe / USA)

| App | What it does |
|---|---|
| **Cleo** (UK) | AI chat assistant that analyzes spending, roasts your habits, sets budgets. Closest to the AI advisor concept. |
| **Monarch Money** (US) | Household budgeting, bill tracking, savings goals, net worth. Clean dashboard UX. |
| **YNAB** (You Need A Budget) | Zero-based budgeting, goal planning, bill reminders. Very popular in US/EU. |
| **Copilot** (US, iOS only) | AI-categorized transactions, spending insights, savings tracking. Premium design. |
| **Emma** (UK) | Multi-bank aggregator, subscription tracker, spending breakdowns. |
| **Plum** (UK) | Auto-saves money based on spending patterns, AI nudges. |
| **Revolut** (UK/EU) | Smart analytics, savings vaults, budget goals — built into the main bank app. |
| **Mint** (US, now dead) | Was the gold standard for budgeting dashboards. Many successors copy its layout. |

---

## CIS / Russia / KZ

| App | What it does |
|---|---|
| **Tinkoff** (Russia) | Most advanced — AI spending insights, receipt scanning, auto-categorization, savings goals, investments. The benchmark for CIS fintech UX. |
| **Sberbank / SberBank Online** (Russia) | Receipt scanning, bill reminders, piggy bank savings, analytics. Massive ecosystem. |
| **Kaspi.kz** (Kazakhstan) | Payments, bill tracking, loans — but analytics is weak. Massive in KZ market. |
| **Halyk Bank** (Kazakhstan) | Basic spending history, bill payments. No real AI features. |
| **Monobank** (Ukraine) | Excellent UX — auto-categorized spending, cashback, savings jars. Very popular. Very clean mobile design. |
| **Raiffeisen** (Russia/EU) | Decent analytics dashboard, budget tools. |
| **Wallet by BudgetBakers** | Manual tracking, popular in CIS because it works without bank integration. |

---

## Key Takeaways for MBank

**Direct competitors in KG/CA region:** Kaspi (KZ) + Halyk — both have weak analytics, no AI. Big gap to fill.

**Design references worth studying:**
- **Monobank** — best mobile UX in CIS, very clean
- **Tinkoff** — most complete feature set
- **Cleo** — best AI chat UX (Western)
- **Copilot** — best visual design for financial data

**Our actual edge:** None of these are localized for Kyrgyzstan — local utilities (Severelectro, Bishkekteploset), KGS currency, Russian/Kyrgyz language-first, and MBank's existing user base.
